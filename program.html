Tembikar

Merupakan sebuah program dari Laskar Negeri.id untuk berbagi pengalaman bersama. Dalam program ini, Laskar Negeri.id akan mengadakan live instagram dengan mengajak teman bincang yang pastinya mempunyai banyak pengalaman dan ilmu yang menarik. Jadi, dengan adanya program ini diharapkan para laskarian punya teman bincang nih untuk sekedar bertanya ataupun sharing hal hal menarik.

Talk With Mentor

Sebuah program pelatihan dari Laskar Negeri.id untuk membantu para guru agar dapat memberikan PJJ yang berkualitas. Kurikulum pelatihan meliputi kemampuan komunikasi, manajemen emosi, adopsi teknologi pembelajaran, kemampuan berpikir kreatif, hingga pembuatan materi pembelajaran.
